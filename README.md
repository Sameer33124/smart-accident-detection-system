# Smart Accident Detection System

An intelligent accident detection and emergency alert system developed using Python and sensor simulation concepts.

## Features
- Detects sudden acceleration and accidents
- Generates emergency alerts
- Graph visualization of sensor data
- Dashboard logging system
- CSV data analysis
- Virtual acceleration simulation

## Technologies Used
- Python
- CSV
- Graph Plotting
- Sensor Simulation
- Embedded System Concepts

## Project Files
- accident_detection_logic.py
- accident_detection_module.py
- accident_detection_with_graphs.py
- dashboard_logging.py
- graphs_from_csv.py
- virtual_acceleration.py
